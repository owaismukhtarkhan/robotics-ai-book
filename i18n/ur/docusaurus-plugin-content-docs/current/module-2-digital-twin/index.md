# ماڈیول 2: ڈیجیٹل ٹوئن

اس ماڈیول میں ہم روبوٹک سمیلیشن (Simulation) کی دنیا کا مطالعہ کریں گے۔ **ڈیجیٹل ٹوئن** ایک جسمانی روبوٹ کا ورچوئل عکس ہوتا ہے جو ہمیں حقیقی دنیا میں تعیناتی سے پہلے الگورتھم کی محفوظ جانچ کرنے کی اجازت دیتا ہے۔

## سمیلیشن پلیٹ فارم: Gazebo بمقابلہ Unity

| خوبی | Gazebo | Unity / Isaac Sim |
| :--- | :--- | :--- |
| **بنیادی استعمال** | فزکس کی توثیق، ٹکراؤ کی جانچ | ہائی فیڈلٹی رینڈرنگ، HRI، تربیت |
| **فزکس انجن** | ODE, Bullet, DART | PhysX |
| **رینڈرنگ** | Ogre | High Definition Render Pipeline (HDRP) |

## سمیلیشن لوپ

ورچوئل روبوٹ اور اس کے ماحول کے درمیان تعامل ایک مسلسل لوپ کی پیروی کرتا ہے:

```mermaid
graph LR
    A[فزکس انجن] -->|اسٹیٹ اپ ڈیٹ| B[سنتھیٹک ڈیٹا (Synthetic Data)]
    B -->|LIDAR/Camera/IMU| C[کنٹرولر/دماغ]
    C -->|احکامات| A
```

## ہفتہ وار جائزہ

### ہفتہ 4: Gazebo فزکس سیٹ اپ
- جوڑوں (Joints) اور رگڑ (Friction) کی ترتیب۔
- سمیلیشن کے ماحول میں روبوٹ کے ڈھانچے کی توثیق۔

### ہفتہ 5: سینسر فیوژن
- ورچوئل سینسرز کا انضمام ([ہارڈ ویئر لیب گائیڈ](/docs/hardware-lab/index.md))۔
- میپنگ کے لیے **سنتھیٹک ڈیٹا** کی تخلیق اور پروسیسنگ۔
